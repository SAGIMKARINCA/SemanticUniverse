# Local Platform Bootstrap

## SemanticUniverse Local Shell

SemanticUniverse icin hemorrhoid uygulamasindan ayri, `localhost` uzerinden acilan bagimsiz bir local shell kuruldu.
Bu yapi, SemanticUniverse'un "Laravel'e tasinmasi" degil; Laravel platformu uzerinde bir `System Ana Yuklenicisi` olarak kurulacagi ana yapinin yerel kabuk gosterimidir.

## Adres
- http://localhost/semantic-universe/

## Ilk Odak
- GodMode super admin shell
- System
- Kaynaklar
- Kaynak Tanimla

## Not
- Bu sayfa public alan degildir
- Public shell daha sonra ayri tasarlanacaktir
- Bu ilk yerel platform, ana SemanticUniverse kabugunu birlikte doldurmak icin olusturuldu
- Dogru kavramsal ifade:
  - Laravel bir framework/platform zeminidir
  - SemanticUniverse ise bu zemin uzerinde kurulan ust sistemdir
  - Ilk kurulum rolu: `System Ana Yuklenicisi`
