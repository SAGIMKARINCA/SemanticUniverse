# Working Model

## Gelistirme Ortamlari

### Local
- hizli iterasyon
- mimari tasarim
- deneysel gelistirme

### Staging
- entegrasyon testi
- sunucu davranisi
- deploy dogrulamasi

### Production
- yalnizca onayli surumler
- izlenebilir deployment
- backup ve rollback stratejisi

## Kaynak Gercegi

Tek kaynak gercegi Git tabanli repo olmalidir.

Yerel gelistirme:
- calisma alani

GitHub:
- resmi versiyon gecmisi

Sunucu:
- deploy edilen calisan surum
