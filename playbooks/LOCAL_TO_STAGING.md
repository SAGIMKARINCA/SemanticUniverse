# Local To Staging

## Amac

Yerelde kurulan bir uygulamayi kontrollu bicimde staging ortamina tasimak.

## Adimlar
- repo guncel ve commitli olmali
- env ayrimi net olmali
- veritabani migration zinciri calismali
- testler yesil olmali
- deploy komutlari dokumante olmali
- staging uzerinde smoke test yapilmali

## Not

Production'a dogrudan local kopyasi ile cikilmaz.
Once Git tabanli, tekrar edilebilir staging akisi kurulur.
